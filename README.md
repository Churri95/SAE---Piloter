# Concession Auto - Site Vitrine

Bienvenue sur le site vitrine de notre **Concession Automobile**. Ce site présente nos véhicules, les services que nous proposons, et nous permet de prendre contact avec nos clients.

## Description

Ce projet a été réalisé dans le cadre d'une présentation de site vitrine pour une concession automobile. Il permet aux visiteurs de découvrir les voitures disponibles à la vente, les services offerts par la concession, et de prendre contact facilement avec l'équipe.

## Structure du projet

- **`index.html`** : Page d'accueil du site.
- **`style.css`** : Feuille de style pour la mise en page du site.
- **`script.js`** : Scripts JavaScript (à ajouter si nécessaire).
- **`images/`** : Dossier contenant toutes les images utilisées sur le site.

## Fonctionnalités

- Navigation simple et claire entre les pages.
- Affichage des véhicules en vente avec des images et des descriptions.
- Formulaire de contact pour les clients intéressés.
